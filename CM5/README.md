# REST

REpresntational State Transfer (REST) is a set of architectural principles for building Web

- ⚠️ not a protocol

## Concepts
### Ressources 
Identified by unique URI
### Methods
- Create
- Read
- Update
- Delete      
...
### Representations
- Data transfer objects (DTO) between the client and the server
- JSON, XML, HTML...
